# modbus_gui

本程序基于 Python 与 Tkinter 开发，提供一个用于 Modbus RTU 协议学习、测试与实验教学的可视化界面。系统能够进行 CRC16(Modbus) 校验、寄存器读写操作，并支持浮点数寄存器处理，适合在无硬件或真实设备环境下进行实验演示。

主要功能：

- 串口参数配置
  -   自动识别可用的串口端口
  -   支持波特率、校验位、停止位设置
  -   可配置从站地址
  -   提供“演示模式”用于无硬件情况
- CRC16 校验工具
  - 输入不含 CRC 的十六进制数据帧
  -  自动计算并附加 Modbus 标准 CRC16
  -  显示完整命令帧（十六进制）
- 功能码 03：读取保持寄存器
  -  指定起始地址与寄存器数量
  -  列表展示十进制与十六进制寄存器值
  -  支持解析连续寄存器为浮点数（多种字节序）
-   能码 06：写单保持寄存器
  -  将整数值写入指定寄存器
  -  显示从站回显帧与 CRC 校验状态
-  写入浮点数（占两个寄存器）
  -  自动将浮点数拆分为两个 16 位寄存器
  -  支持 ABCD / BADC / CDAB / DCBA 四类字节序
  -  依次发送两条 06 功能码写入指令

运行环境：

- Python 3.8+
-  Tkinter（随 Python 默认提供）


安装依赖：

```bash
pip install pyserial
```

运行方式：

```bash
python modbus_tk_gui.py
```

程序将打开一个 Tkinter 窗口，用于执行上述所有实验操作。

附注：

CRC16 使用 Modbus 标准多项式 0xA001

浮点数遵循 IEEE 754 单精度格式

支持虚拟串口与真实串口
